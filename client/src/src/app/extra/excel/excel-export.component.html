Give to Excel (or other files) some REST (JAX-RS)

A common requirement for business application is to provide to the users the possibility to save the data to a document and in many cases this should be a MS Excel document.
With Java we can easily produce an Excel compatible document using the Apache POI library.
But how to transfer this data from the Application Server to the WebClient without creating temporary files?
With JAX-RS we can easily transfer a stream of bytes (ByteArrayInputStream ) via REST service or Servlet to our JS Client
In the JS Client (Angular) you have to store this document in a blog and trigger a download.
