Java, Angular and the Cloud

With this demo I want to show some features of my favorites technologies. Java (Spring, Java EE) for the backend, Angular and TypeScript as frontend and Google Cloud (Kubernetes, Firebase) for the deployment.

The first version of this demo was oriented towards the new features of Java EE 8. Java EE won’t exist anymore with this name and organization, at this time no company supports the specifications of the platform. The Java EE companies are nowadays more interested to sell cloud services than application’s servers licenses.

The new version of this demo is more consistent with the enterprise development and the new trends. It will continuously integrate the new technologies that succeed on the market. The new demo will integrate:

  • Microservices (Spring Boot, Servless functions)
  • Cloud deployment (Kubernetes, Buckets)
  • Web and Mobile (PWA) applications with Angular
