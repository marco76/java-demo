ChatBot using Websockets

This is an example of implementation of WebSocket to create a 'stupid' ChatBot.
The ChatBot answer only to few questions, the goal is to show how the client and the server can establish and carry on a full-duplex communication.
If you ask for a joke the server answer sending multiple delayed messages in the same 'conversation'. Angular is a perfect match to receive ('observe') server notifications.
