The code realizes the following operations:
  • JAX-RS creates a RESTful resource and handles requests
  • BV validates the 'size' parameter, the value must be an Integer between 5 and 20
  • BV validates the 'maxValue' parameter, the value must be an Integer between 5 and 100
  • JAX-RS assigns the parameters to the variables (@QueryParam)
  • Java creates a Random generator with the parameters and streams (IntStream) the result to an array
  • JSON-B builds a JSON object
  • JAX-RS sends the JSON object with the response
